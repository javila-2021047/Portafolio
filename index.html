Habilidades.


Programación

Tengo habilidades en JavaScript y TypeScript para el desarrollo de aplicaciones web dinámicas e interactivas, C# para la creación de aplicaciones robustas y eficientes con .NET, Visual Basic para el desarrollo de soluciones empresariales con facilidad de mantenimiento, y HTML y CSS para el diseño y estructuración de páginas web modernas y accesibles.


Bases de Datos

Tengo habilidades en SQL para la consulta y manipulación de datos, así como experiencia con bases de datos relacionales como MySQL, PostgreSQL y SQL Server. Además, manejo bases de datos NoSQL como MongoDB. Utilizo herramientas como DBeaver para la gestión eficiente y visualización de bases de datos.


Bibliotecas y Frameworks

Puedo trabajar con Node.js para el desarrollo del backend, y con bibliotecas como React.js y Vue.js para la creación de interfaces de usuario dinámicas. También tengo conocimientos básicos de Next.js para aplicaciones web modernas y optimizadas. En cuanto al diseño, utilizo frameworks como Bootstrap y Tailwind CSS para crear interfaces atractivas y responsivas.


Otros

Tengo experiencia en redes, manejo básico de los paquetes de Office, y uso herramientas como Git para el control de versiones y Postman para probar y consumir APIs.
